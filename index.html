Project Origin

ที่มาของ No.AI

โปรเจค No.AI เริ่มจากคำถามของครู สกร. คนหนึ่ง ที่คิดว่ามันมี prompt ด่วนๆ สำหรับงานครู สกร.ที่จะนำไปใช้แบบด่วนๆหรือไม่...ซึ่งต้องไม่ใช่คำสั่งกว้าง ๆ อารมณ์แบบ คิด คัดลอก วาง ไรงี้....โอเคแหล่ะว่ามันต้องมาช่วยลดเวลางานเอกสาร งานสอน และงานพัฒนาผู้เรียน รวมถึงงานอื่นๆ ที่ไม่น่าไปเสียเวลาทำมากนัก....

แนวคิดของเว็บคือให้ครูเลือก “งานที่ต้องทำ” ก่อน แล้วระบบช่วยประกอบ prompt ให้ (เบื้องต้นนะ...ใครที่โปรจัดๆ ผ่านไปเลยดีกว่า)
